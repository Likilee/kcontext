## 1. Brand Essence

### 1.1 Brand Name & Tagline

- Name: kcontext (소문자 표기 지향. 무겁고 권위적인 교육 플랫폼이 아닌, 빠르고 매끄럽게 작동하는 모던 유틸리티 툴임을 시각적으로 강조)
- Main Tagline: Real Korean, Right in Context.
- Sub Tagline: Stop memorizing textbooks. Start experiencing conversations.

### 1.2 Mission & Vision

- Mission: 전 세계 한국어 학습자들이 교과서에 갇힌 죽은 한국어(Textbook Korean)에서 벗어나, 원어민이 실제로 사용하는 생생한 뉘앙스와 맥락을 0.1초 만에 탐색하고 흡수하도록 연결한다.
- Vision: 가장 빠르고 정확한 '글로벌 한국어 영상 문맥 검색 표준 엔진'이 되어, 진성 학습자와 튜터를 연결하는 필수 어학 인프라로 자리 잡는다.

### 1.3 Core Values

1. Authentic (진정성): 가공된 예문이나 기계음(TTS)을 철저히 배제하고, 뉴스/예능/드라마 등 100% 원어민의 실제 발화 데이터와 수동 자막만을 제공한다.
2. Frictionless (무마찰): 학습의 흐름을 끊는 로딩, 복잡한 네비게이션, 기계적인 에러, 완벽한 싱크에 대한 집착 등 모든 인지적/시각적 마찰을 0으로 만든다.
3. Empowering (주도성): 지시하거나 가르치려 들지 않고, 학습자와 강사가 스스로 언어를 수집하고 활용할 수 있는 강력한 통제권(북마크, 컬렉션, 내보내기)을 제공한다.

## 2. Brand Persona

kcontext를 의인화한다면 "스마트하고 센스 있는 서울의 로컬 친구 겸 유능한 튜터링 조수"이다.

- 권위적이거나 학술적인 언어학 교사가 아니다. 복잡한 문법 용어를 늘어놓기보다 "한국인들은 실제로 이럴 때 이 말을 써!"라며 직관적인 영상 증거를 즉시 찾아 보여주는 실용주의자다.
- K-Pop 슬랭을 과도하게 남발하는 가벼운 캐릭터가 아니며, 글로벌 SaaS 도구로서의 빠르고 정확한 신뢰감을 밑바탕에 둔다.

## 3. Tone of Voice

글로벌 유저를 타겟으로 하므로 서비스의 기본 UI 및 마케팅 언어는 영어(Plain English)를 기준으로 작성하며, 명료하고 직관적인 SaaS 스타일의 문장력을 구사한다.

### 3.1 Core Principles

- Clear & Action-Oriented (명확하고 행동 중심적으로): 수동태나 장황한 설명을 피하고, 사용자가 다음 행동을 하도록 직관적인 동사를 전면에 배치한다.
- Encouraging & Empathetic (격려하고 공감하는): 외국어 학습의 좌절감을 이해한다. 검색 결과가 없거나 오타가 발생했을 때, 기계적인 에러 메시지 대신 부드러운 대안을 제시하여 이탈을 방지한다.
- Smart, but not Academic (똑똑하지만 지루하지 않게): 문법 용어(명사형 어미, 관형격 조사 등)를 남발하지 않고 직관적인 쓰임새와 가치 전달에 집중한다.

### 3.2 UX Microcopy Guidelines

- Search Placeholder (검색창 힌트):
백지 증후군(Blank Canvas Syndrome) 방지를 위해 구체적 예시 제공.
    - [Bad] Search here...
    - [Good] Search for real Korean (e.g. 진짜 행복해요, 어쩔티비)
- Empty State (검색 결과 없음):
    - [Bad] 0 results found. / Error 404
    - [Good] Hmm, even native speakers rarely use this exact phrase. Try searching for a shorter keyword!
- Player Controls (플레이어 제어):
기계적인 단어 대신 '문맥(Context)'을 강조.
    - [Bad] Repeat / Loop Video
    - [Good] Replay context
- Call to Action (유료 전환 및 기능 유도):
기능 자체가 아닌 유저가 얻게 될 '가치(시간 절약, 효율)'를 강조.
    - [B2C] "Stop copy-pasting. Export this exact context to Anki in one click."
    - [B2B2C] "Save 3 hours of lesson prep. Create your first video collection for students."

## 4. Visual Identity

콘텐츠(유튜브 영상과 자막) 자체가 주인공이 되어야 하므로, UI는 시각적 노이즈를 극도로 통제하여 스스로를 투명하게 지우는 캔버스(Invisible UI) 역할을 한다.

### 4.1 Color Palette

- Primary Action Color (Highlighter Yellow - `#FDE047`):
    - 의미: 종이책에 긋는 '형광펜'의 아날로그적 경험을 디지털로 치환. 브랜드의 핵심 가치인 '내가 찾은 바로 그 맥락'을 직관적으로 상징.
    - 적용: 검색된 자막 텍스트의 배경색(Static Highlight) 및 핵심 CTA 버튼 등 시선이 집중되어야 하는 극소수의 영역에만 제한적으로 사용한다.
- Base Colors (Slate / Zinc):
    - 시각적 정보량이 많은 영상 플레이어를 감싸기 위해, UI 배경은 시선을 뺏지 않는 무채색(Off-White, Dark Gray)을 사용한다. 영상 몰입도를 극대화하는 다크 모드(Dark Mode) 지원을 기본으로 고려한다.
- Text Colors (Solid Uniformity):
    - 시각적 인지 부조화를 유발하는 동적 글자색 변경(소프트 포커스)이나 불필요한 텍스트 애니메이션은 전면 배제한다.
    - 모든 하단 자막 텍스트는 단일한 고대비 색상(예: 다크 모드 시 Solid White, 라이트 모드 시 Ink Black)으로 렌더링을 고정한다. 오직 검색어 영역에만 노란색 배경(Static Highlight)이 고정되도록 하여, 네이티브 유튜브 자막(CC)과 동일한 인지적 편안함을 유지한다.

### 4.2 Typography

다국어 환경에서 시스템 도구로서의 신뢰감과 자막 가독성을 동시에 확보한다.

- UI / English Text: `Inter` 또는 `Geist` (가독성이 검증된 모던 글로벌 산세리프 폰트로 IT 검색 툴의 신뢰감 제공)
- Subtitle / Korean Text: `Pretendard` (한국어 텍스트 렌더링 시 자간, 띄어쓰기 판독, 굵기 밸런스가 가장 우수한 폰트)
- Rule: 자막 뷰어 영역의 폰트 사이즈는 일반 웹 텍스트보다 1.5배 이상 크게 설정하고 줄 간격(Line-height)을 여유롭게 확보하여, 영상과 텍스트를 번갈아 보는 시각적 피로도를 최소화한다.
